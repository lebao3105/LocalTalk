# LocalTalk

Transfer your files between your devices!

Supports:

* Windows Phone 8.1 (on-going)
* UWP 10 (on-going)

## Capabilities

* Device discovery :white_check_mark:?
* HTTP(s) usage - not only Multicast :x:
